# InsurAI Project

InsurAI is an AI-based insurance platform designed to analyze user data and provide smart policy recommendations.

## Features
- AI-based UI
- User dashboard
- Risk analysis system
- Policy recommendations

## Tech Stack
- Frontend: React (TypeScript)
- Backend: Node.js (Express)

## Project Structure
- frontend/ → UI code
- backend/ → API server
- presentations/ → PPT files

## APIs
- GET /api/policies
- POST /api/user
- POST /api/risk

## Note
This project includes frontend UI and a basic backend for demonstration.
